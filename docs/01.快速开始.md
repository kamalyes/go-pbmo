# 01. 快速开始

## 安装

```bash
go get github.com/kamalyes/go-pbmo
```

## 最小示例

```go
package main

import (
    "fmt"
    pbmo "github.com/kamalyes/go-pbmo"
)

type UserPB struct {
    Id    uint64
    Name  string
    Email string
    Age   int32
}

type UserModel struct {
    ID    uint64 `pbmo:"Id"`
    Name  string
    Email string
    Age   int
}

func main() {
    converter := pbmo.NewBidiConverter(UserPB{}, UserModel{})

    // PB -> Model
    pb := UserPB{Id: 1, Name: "张三", Email: "test@example.com", Age: 25}
    var model UserModel
    if err := converter.ConvertPBToModel(pb, &model); err != nil {
        panic(err)
    }
    fmt.Printf("Model: %+v\n", model)

    // Model -> PB
    model2 := UserModel{ID: 2, Name: "李四", Age: 30}
    var pb2 UserPB
    if err := converter.ConvertModelToPB(model2, &pb2); err != nil {
        panic(err)
    }
    fmt.Printf("PB: %+v\n", pb2)
}
```

## 核心 API 一览

| 函数 | 说明 |
|------|------|
| `NewBidiConverter(pbType, modelType, ...Option)` | 创建双向转换器 |
| `converter.ConvertPBToModel(pb, &model)` | PB → Model |
| `converter.ConvertModelToPB(model, &pb)` | Model → PB |
| `converter.Validate(data)` | 校验数据 |
| `converter.RegisterTransformer(field, fn)` | 注册字段转换器 |
| `converter.WithFieldMapping(modelField, pbField)` | 设置字段映射 |

## 导入约定

```go
import pbmo "github.com/kamalyes/go-pbmo"
```
