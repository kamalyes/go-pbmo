# 05. 批量转换

当需要转换列表数据时，使用批量转换 API 可以简化代码并提高效率

## 基础批量转换

### BatchConvertPBToModel

```go
pbs := []UserPB{
    {Id: 1, Name: "张三"},
    {Id: 2, Name: "李四"},
}

var models []UserModel
err := converter.BatchConvertPBToModel(pbs, &models)
```

### BatchConvertModelToPB

```go
models := []UserModel{
    {ID: 1, Name: "张三"},
    {ID: 2, Name: "李四"},
}

var pbs []UserPB
err := converter.BatchConvertModelToPB(models, &pbs)
```

## 安全批量转换

安全批量转换不会因为单个元素失败而中断整个流程，而是收集所有结果：

### SafeBatchConvertPBToModel

```go
result := converter.SafeBatchConvertPBToModel(pbs, &models)

fmt.Printf("成功: %d, 失败: %d\n", result.SuccessCount, result.FailureCount)

for _, item := range result.Results {
    if item.Success {
        fmt.Printf("[%d] 成功: %+v\n", item.Index, item.Value)
    } else {
        fmt.Printf("[%d] 失败: %v\n", item.Index, item.Error)
    }
}
```

### SafeBatchConvertModelToPB

```go
result := converter.SafeBatchConvertModelToPB(models, &pbs)
```

## BatchResult 结构

```go
type BatchResult struct {
    SuccessCount int         // 成功数量
    FailureCount int         // 失败数量
    Results      []BatchItem // 每个元素的转换结果
}

type BatchItem struct {
    Index   int         // 索引
    Success bool        // 是否成功
    Value   interface{} // 转换后的值
    Error   error       // 错误信息
}
```

## 指针切片支持

批量转换自动识别目标切片是否为指针类型：

```go
// 值切片
var models []UserModel
converter.BatchConvertPBToModel(pbs, &models)

// 指针切片
var modelPtrs []*UserModel
converter.BatchConvertPBToModel(pbs, &modelPtrs)
```

## 错误处理

### 基础批量 — 遇错即停

`BatchConvertPBToModel` / `BatchConvertModelToPB` 在遇到第一个错误时立即返回：

```go
err := converter.BatchConvertPBToModel(pbs, &models)
if err != nil {
    // err 为 BatchError，包含失败元素的索引和原因
    if batchErr, ok := err.(*pbmo.BatchError); ok {
        fmt.Printf("批量转换失败: %v\n", batchErr)
    }
}
```

### 安全批量 — 收集所有结果

`SafeBatchConvertPBToModel` / `SafeBatchConvertModelToPB` 不会中断，每个元素的结果独立记录：

```go
result := converter.SafeBatchConvertPBToModel(pbs, &models)
if result.FailureCount > 0 {
    for _, item := range result.Results {
        if !item.Success {
            log.Printf("元素 %d 转换失败: %v", item.Index, item.Error)
        }
    }
}
```
