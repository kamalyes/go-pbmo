# 05. 批量转换

当需要转换列表数据时，使用批量转换 API 可以简化代码并提高效率

## 泛型批量转换（推荐）

泛型批量 API 直接操作 `[]*T` 类型切片，类型安全、调用简洁：

### ToPBs — 批量 Model → PB

```go
models := []*UserModel{
    {ID: 1, Name: "张三"},
    {ID: 2, Name: "李四"},
}

pbs, err := pbmo.ToPBs[UserModel, UserPB](models)
// pbs: []*UserPB{{Id: 1, Name: "张三"}, {Id: 2, Name: "李四"}}
```

### FromPBs — 批量 PB → Model

```go
pbs := []*UserPB{
    {Id: 1, Name: "张三"},
    {Id: 2, Name: "李四"},
}

models, err := pbmo.FromPBs[UserPB, UserModel](pbs)
// models: []*UserModel{{ID: 1, Name: "张三"}, {ID: 2, Name: "李四"}}
```

### SafeToPBs / SafeFromPBs — 安全批量转换

安全批量转换不会因为单个元素失败而中断整个流程，而是收集所有结果：

```go
models := []*UserModel{
    {ID: 1, Name: "张三"},
    nil,  // 不会中断
    {ID: 3, Name: "王五"},
}

pbs, result := pbmo.SafeToPBs[UserModel, UserPB](models)

fmt.Printf("成功: %d, 失败: %d\n", result.SuccessCount, result.FailureCount)
// 成功: 2, 失败: 1

for _, item := range result.Results {
    if item.Success {
        fmt.Printf("[%d] 成功: %+v\n", item.Index, item.Value)
    } else {
        fmt.Printf("[%d] 失败: %v\n", item.Index, item.Error)
    }
}
```

### 空切片和 nil 切片

```go
// 空切片 — 返回空切片，无错误
pbs, err := pbmo.ToPBs[UserModel, UserPB]([]*UserModel{})

// nil 切片 — 返回空切片，无错误
var models []*UserModel
pbs, err = pbmo.ToPBs[UserModel, UserPB](models)
```

## 传统批量转换（Deprecated）

> ⚠️ 传统批量 API 已标记为 **Deprecated**，建议迁移到泛型 API

### BatchConvertPBToModel

```go
// Deprecated: 使用 FromPBs[P, M] 替代
pbs := []UserPB{{Id: 1, Name: "张三"}, {Id: 2, Name: "李四"}}
var models []UserModel
err := converter.BatchConvertPBToModel(pbs, &models)
```

### BatchConvertModelToPB

```go

var pbs []UserPB
err := converter.BatchConvertModelToPB(models, &pbs)
```

### SafeBatchConvertPBToModel / SafeBatchConvertModelToPB

```go
// Deprecated: 使用 SafeFromPBs[P, M] / SafeToPBs[M, P] 替代
result := converter.SafeBatchConvertPBToModel(pbs, &models)
result = converter.SafeBatchConvertModelToPB(models, &pbs)
```

## BatchResult 结构

```go
type BatchResult struct {
    SuccessCount int         // 成功数量
    FailureCount int         // 失败数量
    Results      []BatchItem // 每个元素的转换结果
}

type BatchItem struct {
    Index   int         // 索引
    Success bool        // 是否成功
    Value   interface{} // 转换后的值
    Error   error       // 错误信息
}
```

## 错误处理

### 泛型批量 — 遇错即停

`ToPBs` / `FromPBs` 在遇到第一个错误时立即返回：

```go
pbs, err := pbmo.ToPBs[UserModel, UserPB](models)
if err != nil {
    // err 为 BatchError，包含失败元素的索引和原因
    if batchErr, ok := err.(*pbmo.BatchError); ok {
        fmt.Printf("批量转换失败: %v\n", batchErr)
    }
}
```

### 泛型安全批量 — 收集所有结果

`SafeToPBs` / `SafeFromPBs` 不会中断，每个元素的结果独立记录：

```go
pbs, result := pbmo.SafeToPBs[UserModel, UserPB](models)
if result.FailureCount > 0 {
    for _, item := range result.Results {
        if !item.Success {
            log.Printf("元素 %d 转换失败: %v", item.Index, item.Error)
        }
    }
}
```

## 迁移对照表

| 旧 API（Deprecated） | 新 API（推荐） |
|----------------------|---------------|
| `converter.BatchConvertModelToPB(models, &pbs)` | `ToPBs[M, P](models)` |
| `converter.BatchConvertPBToModel(pbs, &models)` | `FromPBs[P, M](pbs)` |
| `converter.SafeBatchConvertModelToPB(models, &pbs)` | `SafeToPBs[M, P](models)` |
| `converter.SafeBatchConvertPBToModel(pbs, &models)` | `SafeFromPBs[P, M](pbs)` |
