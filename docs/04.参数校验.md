# 04. 参数校验

go-pbmo 内置字段级校验器（`Validator`），支持必填、长度、范围、正则、自定义等校验规则

## FieldRule 校验规则

```go
type FieldRule struct {
    Name      string                  // 字段名
    Required  bool                    // 是否必填
    MinLen    int                     // 最小长度
    MaxLen    int                     // 最大长度
    Min       float64                 // 最小值
    Max       float64                 // 最大值
    Pattern   string                  // 正则表达式
    Custom    func(interface{}) error // 自定义校验函数
    Transform TransformerFunc         // 校验通过后的转换函数
}
```

## 注册校验规则

### 通过 BidiConverter

```go
converter := pbmo.NewBidiConverter(UserPB{}, UserModel{})

converter.RegisterValidationRules("UserModel",
    pbmo.FieldRule{Name: "Name", Required: true, MinLen: 1, MaxLen: 50},
    pbmo.FieldRule{Name: "Email", Required: true, Pattern: `^[^@]+@[^@]+\.[^@]+$`},
    pbmo.FieldRule{Name: "Age", Min: 0, Max: 150},
)
```

### 通过独立 Validator

```go
validator := pbmo.NewValidator()

validator.RegisterRules("UserModel",
    pbmo.FieldRule{Name: "Name", Required: true},
    pbmo.FieldRule{Name: "Age", Min: 0, Max: 150},
)

// 批量注册
validator.RegisterBatch(map[string][]pbmo.FieldRule{
    "UserModel": {
        {Name: "Name", Required: true},
    },
    "ProductModel": {
        {Name: "Name", Required: true},
        {Name: "Price", Min: 0.01},
    },
})
```

## 执行校验

### 转换时自动校验

启用 `WithValidation(true)` 选项后，`ConvertPBToModel` 会在转换完成后自动执行校验：

```go
converter := pbmo.NewBidiConverter(UserPB{}, UserModel{},
    pbmo.WithValidation(true),
)

converter.RegisterValidationRules("UserModel",
    pbmo.FieldRule{Name: "Name", Required: true},
)

var model UserModel
err := converter.ConvertPBToModel(pb, &model)
// 如果校验失败，err 为 ValidationErrors
```

### 手动校验

```go
err := converter.Validate(&model)
// 或
err := validator.Validate(&model)
```

## ValidationErrors

校验失败时返回 `ValidationErrors`（`[]ValidationError`）：

```go
type ValidationError struct {
    Field   string // 字段名
    Message string // 错误信息
}

type ValidationErrors []ValidationError
```

```go
err := validator.Validate(data)
if err != nil {
    if validationErrs, ok := err.(pbmo.ValidationErrors); ok {
        for _, e := range validationErrs {
            fmt.Printf("字段 %s: %s\n", e.Field, e.Message)
        }
    }
}
```

### ValidationErrors 方法

| 方法 | 说明 |
|------|------|
| `Error() string` | 实现 error 接口 |
| `HasErrors() bool` | 是否有错误 |
| `Fields() []string` | 获取所有错误字段名 |

## 校验规则详解

### Required 必填

```go
pbmo.FieldRule{Name: "Email", Required: true}
```

- 指针字段为 nil 时判定为空
- 非指针字段为零值时判定为空

### MinLen / MaxLen 长度校验

仅对 `string` 类型字段生效：

```go
pbmo.FieldRule{Name: "Name", MinLen: 1, MaxLen: 100}
```

### Min / Max 范围校验

仅对数值类型字段生效：

```go
pbmo.FieldRule{Name: "Age", Min: 0, Max: 150}
pbmo.FieldRule{Name: "Price", Min: 0.01}
```

### Pattern 正则校验

仅对 `string` 类型字段生效：

```go
pbmo.FieldRule{Name: "Email", Pattern: `^[^@]+@[^@]+\.[^@]+$`}
pbmo.FieldRule{Name: "Phone", Pattern: `^\d{11}$`}
```

### Custom 自定义校验

```go
pbmo.FieldRule{
    Name: "Status",
    Custom: func(val interface{}) error {
        status := val.(int32)
        if status < 0 || status > 3 {
            return fmt.Errorf("状态值 %d 无效，有效范围 0-3", status)
        }
        return nil
    },
}
```

### Transform 校验后转换

校验通过后自动应用转换函数：

```go
pbmo.FieldRule{
    Name:     "Email",
    Required: true,
    Transform: func(val interface{}) interface{} {
        return strings.ToLower(val.(string))
    },
}
```

## 校验器管理 API

| 方法 | 说明 |
|------|------|
| `RegisterRules(structName, rules...)` | 注册规则 |
| `RegisterBatch(map)` | 批量注册 |
| `Validate(data)` | 执行校验 |
| `ValidateWithTransform(data)` | 校验并转换 |
| `GetRules(structName)` | 获取规则 |
| `HasRules(structName)` | 检查是否有规则 |
| `RemoveRules(structName)` | 移除规则 |
| `Clear()` | 清空所有规则 |

## 与中间件集成

在 go-rpc-gateway 中，`PBValidationMiddleware` 使用 `go-pbmo.Validator` 作为底层校验引擎：

```go
middleware := NewPBValidationMiddleware()

// 注册校验规则
middleware.RegisterRules("CreateUserRequest",
    gopbmo.FieldRule{Name: "Name", Required: true, MinLen: 1},
    gopbmo.FieldRule{Name: "Email", Required: true, Pattern: `^[^@]+@[^@]+$`},
)

// gRPC 请求会自动校验
```
