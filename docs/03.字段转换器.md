# 03. 字段转换器

字段转换器（Transformer）允许你在转换过程中对特定字段应用自定义逻辑，例如类型适配、数据格式化等

## TransformerFunc

```go
type TransformerFunc func(interface{}) interface{}
```

输入原始字段值，输出转换后的值

## 注册字段转换器

### 单个注册

```go
converter := pbmo.NewBidiConverter(UserPB{}, UserModel{})

converter.RegisterTransformer("Status", func(val interface{}) interface{} {
    // PB int32 状态码 -> Model 字符串状态
    status := val.(int32)
    switch status {
    case 0:
        return "unknown"
    case 1:
        return "active"
    case 2:
        return "inactive"
    default:
        return "unknown"
    }
})
```

### 批量注册

```go
converter.GetTransformers().RegisterBatch(map[string]pbmo.TransformerFunc{
    "Status": statusTransformer,
    "Role":   roleTransformer,
})
```

## 转换器执行时机

转换器在**字段名映射之后、类型转换之前**执行：

```
源字段 → 字段名映射 → 应用转换器 → 类型转换 → 目标字段
```

这意味着转换器接收的是**源字段值**，输出值需要能被后续类型转换处理

## 转换器注册表 API

`TransformerRegistry` 提供完整的管理能力：

| 方法 | 说明 |
|------|------|
| `Register(field, fn)` | 注册转换器 |
| `RegisterBatch(map)` | 批量注册 |
| `Lookup(field)` | 查找转换器 |
| `Apply(field, value)` | 应用转换器 |
| `Has(field)` | 检查是否存在 |
| `Remove(field)` | 移除转换器 |
| `Clear()` | 清空所有 |
| `Fields()` | 获取所有已注册字段名 |
| `Count()` | 获取数量 |
| `Clone()` | 克隆注册表 |

## 常见场景

### 枚举值转换

```go
converter.RegisterTransformer("Type", func(val interface{}) interface{} {
    pbType := val.(int32)
    return modelTypeFromPB(pbType)
})
```

### 字符串格式化

```go
converter.RegisterTransformer("Phone", func(val interface{}) interface{} {
    phone := val.(string)
    return strings.ReplaceAll(phone, "-", "")
})
```

### 时间格式转换

```go
converter.RegisterTransformer("CreatedAt", func(val interface{}) interface{} {
    // 如果 PB 用 string 存储时间，转为 time.Time
    if s, ok := val.(string); ok {
        t, _ := time.Parse("2006-01-02", s)
        return t
    }
    return val
})
```

## 注意事项

- 转换器是**按字段名**注册的，不是按结构体类型
- 同名字段会共享转换器，注意命名冲突
- 转换器函数不可比较（non-comparable），内部使用 `sync.Map` 存储
