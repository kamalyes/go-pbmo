# 10. 时间转换

go-pbmo 自动处理 `time.Time` ↔ `*timestamppb.Timestamp` 之间的转换，无需手动编写转换代码

## 自动时间转换

默认启用，在创建转换器时自动开启：

```go
converter := pbmo.NewBidiConverter(EventPB{}, EventModel{})
// AutoTimeConversion 默认为 true
```

### PB → Model

```go
type EventPB struct {
    Id        uint64
    Name      string
    CreatedAt *timestamppb.Timestamp
    UpdatedAt *timestamppb.Timestamp
}

type EventModel struct {
    ID        uint64 `pbmo:"Id"`
    Name      string
    CreatedAt time.Time
    UpdatedAt time.Time
}

converter := pbmo.NewBidiConverter(EventPB{}, EventModel{})

var model EventModel
converter.ConvertPBToModel(pb, &model)
// CreatedAt 和 UpdatedAt 自动从 *timestamppb.Timestamp 转为 time.Time
```

### Model → PB

```go
var pb EventPB
converter.ConvertModelToPB(model, &pb)
// CreatedAt 和 UpdatedAt 自动从 time.Time 转为 *timestamppb.Timestamp
```

## 禁用自动时间转换

```go
converter := pbmo.NewBidiConverter(EventPB{}, EventModel{},
    pbmo.WithAutoTimeConversion(false),
)
```

## 手动时间转换

go-pbmo 提供独立的时间转换函数：

```go
// time.Time -> *timestamppb.Timestamp
timestamp := pbmo.TimeToTimestamp(time.Now())

// *timestamppb.Timestamp -> time.Time
t := pbmo.TimestampToTime(timestamp)

// 安全转换（处理 nil）
t, ok := pbmo.SafeTimestampToTime(nil) // ok == false, t 为零值
```

## 时间转换函数一览

| 函数 | 说明 |
|------|------|
| `TimeToTimestamp(t time.Time) *timestamppb.Timestamp` | Time → Timestamp |
| `TimestampToTime(ts *timestamppb.Timestamp) time.Time` | Timestamp → Time |
| `SafeTimestampToTime(ts *timestamppb.Timestamp) (time.Time, bool)` | 安全转换，nil 返回零值 |

## 注意事项

- 自动时间转换仅处理 `time.Time` ↔ `*timestamppb.Timestamp` 的配对
- 如果 PB 使用 `string` 存储时间，需要通过字段转换器处理
- `nil` 的 `*timestamppb.Timestamp` 会转为 `time.Time` 零值
- `time.Time` 零值会转为 `nil` 的 `*timestamppb.Timestamp`
