# 07. 枚举映射

PB 和 Model 之间经常使用不同的枚举值表示同一语义 go-pbmo 提供三种枚举映射方案

## EnumMapper（int32 枚举）

适用于 PB 枚举（int32）与 Model 枚举之间的映射：

```go
mapper := pbmo.NewEnumMapper()

// 链式添加映射
mapper.
    AddMapping(0, 0). // UNKNOWN
    AddMapping(1, 1). // ACTIVE
    AddMapping(2, 2). // INACTIVE

// 批量添加
mapper.AddMappings([][2]int32{
    {0, 0}, {1, 1}, {2, 2},
})

// 正向映射
modelStatus := mapper.Map(pbStatus, 0) // 第 2 个参数为默认值

// 反向映射
pbStatus := mapper.ReverseMap(modelStatus, 0)
```

### MapEnum（反射方式）

使用反射自动处理不同枚举类型：

```go
result, err := mapper.MapEnum(pbStatus, reflect.TypeOf(ModelStatus(0)), ModelStatus(0))
```

### ConvertEnum（泛型便捷方法）

```go
modelStatus := pbmo.ConvertEnum(mapper, pbStatus, ModelStatus(0))
```

## GenericEnumMapper（泛型枚举）

适用于任意可比较类型的枚举映射：

```go
mapper := pbmo.NewGenericEnumMapper[PBStatus, ModelStatus](ModelStatusUnknown)

// 链式注册
mapper.
    Register(PBStatusActive, ModelStatusActive).
    Register(PBStatusInactive, ModelStatusInactive)

// 批量注册
mapper.RegisterBatch(map[PBStatus]ModelStatus{
    PBStatusActive:   ModelStatusActive,
    PBStatusInactive: ModelStatusInactive,
})

// 正向映射
modelStatus := mapper.Map(pbStatus)

// 带默认值映射
modelStatus := mapper.MapWithDefault(pbStatus, ModelStatusUnknown)

// 反向映射
pbStatus, exists := mapper.ReverseMap(modelStatus)

// 批量映射
modelStatuses := mapper.MapSlice([]PBStatus{1, 2, 3})
```

## AutoEnumConverter（自动枚举转换器）

封装 `GenericEnumMapper`，提供更简洁的 API：

```go
converter := pbmo.NewAutoEnumConverter[PBStatus, ModelStatus](ModelStatusUnknown)

converter.AutoRegister(map[PBStatus]ModelStatus{
    PBStatusActive:   ModelStatusActive,
    PBStatusInactive: ModelStatusInactive,
})

// 转换
modelStatus := converter.Convert(pbStatus)

// 反向转换
pbStatus, exists := converter.ConvertBack(modelStatus)

// 批量转换
modelStatuses := converter.ConvertSlice(pbStatuses)

// 带默认值转换
modelStatus := converter.ConvertWithDefault(pbStatus, ModelStatusUnknown)
```

## 性能

枚举映射是零内存分配的热路径操作：

| 场景 | 耗时 | 内存分配 |
|------|------|----------|
| EnumMapper.Map | ~33ns | 0 B |
| GenericEnumMapper.Map | ~36ns | 0 B |
| AutoEnumConverter.Convert | ~52ns | 0 B |

## 选择指南

| 场景 | 推荐方案 |
|------|----------|
| PB int32 ↔ Model int32 | `EnumMapper` |
| 任意类型枚举映射 | `GenericEnumMapper` |
| 简洁 API + 自动注册 | `AutoEnumConverter` |
| 需要批量映射 | `GenericEnumMapper.MapSlice` 或 `AutoEnumConverter.ConvertSlice` |
