# 08. 脱敏转换

go-pbmo 集成 `go-toolbox/desensitize`，在 PB ↔ Model 转换过程中自动应用脱敏规则

## DesensitizeConverter

```go
converter := pbmo.NewDesensitizeConverter(UserPB{}, UserModel{},
    pbmo.WithDesensitize(true),
)
```

### PB → Model 并脱敏

```go
var model UserModel
err := converter.ConvertPBToModelWithDesensitize(pb, &model)
// 转换完成后自动脱敏
```

### Model → PB 并脱敏

```go
var pb UserPB
err := converter.ConvertModelToPBWithDesensitize(model, &pb)
// 先脱敏再转换
```

## 脱敏 Tag

在 Model 结构体上使用 `desensitize` tag 声明脱敏规则：

```go
type UserModel struct {
    ID      uint64
    Name    string
    Phone   string `desensitize:"phone"`    // 手机号脱敏: 138****5678
    Email   string `desensitize:"email"`    // 邮箱脱敏: t***@example.com
    IDCard  string `desensitize:"idCard"`   // 身份证脱敏: 310***********5678
    BankCard string `desensitize:"bankCard"` // 银行卡脱敏: 6222****5678
}
```

## 开关控制

```go
converter.SetEnabled(true)  // 启用脱敏
converter.SetEnabled(false) // 禁用脱敏

if converter.IsEnabled() {
    // 脱敏已启用
}
```

## 独立脱敏函数

不通过 DesensitizeConverter，直接使用包级函数：

### ApplyDesensitization

对任意对象应用脱敏（基于 tag 自动发现）：

```go
err := pbmo.ApplyDesensitization(&model)
```

### DesensitizeField

对单个字段值脱敏：

```go
phone := pbmo.DesensitizeField("13856785678", desensitize.Phone)
// "138****5678"
```

### RegisterDesensitizer

注册自定义脱敏器：

```go
pbmo.RegisterDesensitizer("custom", MyCustomDesensitizer{})
```

## 常见脱敏类型

| Tag 值 | 说明 | 示例输入 | 示例输出 |
|--------|------|----------|----------|
| `phone` | 手机号 | 13856785678 | 138****5678 |
| `email` | 邮箱 | test@example.com | t***@example.com |
| `idCard` | 身份证 | 310101199001015678 | 310***********5678 |
| `bankCard` | 银行卡 | 6222025678565678 | 6222****5678 |
| `name` | 姓名 | 张三丰 | 张** |
| `address` | 地址 | 北京市朝阳区xxx | 北京市*** |
