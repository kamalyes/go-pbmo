# 11. 选项模式

go-pbmo 支持两种配置方式：**Functional Options**（创建时配置）和**链式调用**（创建后配置），支持灵活组合选项

## 完整选项列表

| Functional Option | 链式方法 | 类型 | 默认值 | 说明 |
|-------------------|----------|------|--------|------|
| `pbmo.WithAutoTimeConversion(bool)` | `.WithAutoTimeConversion(bool)` | bool | `true` | 自动时间转换 |
| `pbmo.WithFieldMapping(model, pb)` | `.WithFieldMapping(model, pb)` | string | - | 单个字段映射 |
| `pbmo.WithFieldMappings(map)` | `.WithFieldMappings(map)` | map | - | 批量字段映射 |
| `pbmo.WithTagMapping(bool)` | `.WithTagMapping(bool)` | bool | `true` | 启用 struct tag 映射 |
| `pbmo.WithTagName(name)` | `.WithTagName(name)` | string | `"pbmo"` | struct tag 名称 |
| `pbmo.WithValidation(bool)` | `.WithValidation(bool)` | bool | `false` | 启用校验 |
| `pbmo.WithDesensitize(bool)` | `.WithDesensitize(bool)` | bool | `false` | 启用脱敏 |
| `pbmo.WithSafeMode(bool)` | `.WithSafeMode(bool)` | bool | `false` | 启用安全模式 |
| `pbmo.WithConcurrency(n)` | `.WithConcurrency(n)` | int | `0` | 并发数 |
| `pbmo.WithBatchSize(size)` | `.WithBatchSize(size)` | int | `100` | 批处理大小 |
| `pbmo.WithTimeout(duration)` | `.WithTimeout(duration)` | Duration | `30s` | 超时时间 |

## 方式一：Functional Options（创建时）

```go
converter := pbmo.NewBidiConverter(UserPB{}, UserModel{},
    pbmo.WithAutoTimeConversion(true),
    pbmo.WithValidation(true),
    pbmo.WithFieldMapping("ID", "Id"),
    pbmo.WithDesensitize(true),
)
```

## 方式二：链式调用（创建后）

```go
converter := pbmo.NewBidiConverter(UserPB{}, UserModel{}).
    WithAutoTimeConversion(true).
    WithValidation(true).
    WithFieldMapping("ID", "Id").
    WithDesensitize(true)
```

链式调用更直观，所有方法返回 `*BidiConverter`，可以无限链式组合

两种方式可以混用：创建时传入基础选项，创建后通过链式调用补充配置

```go
converter := pbmo.NewBidiConverter(UserPB{}, UserModel{},
    pbmo.WithAutoTimeConversion(true),
).WithValidation(true).
    WithFieldMapping("ID", "Id")
```

## Options 结构体

```go
type Options struct {
    AutoTimeConversion bool              // 自动时间转换开关
    FieldMapping       map[string]string // 字段名映射: Model字段名 -> PB字段名
    TagMappingEnabled  bool              // 是否启用 struct tag 自动映射
    TagName            string            // struct tag 名称（默认 pbmo）
    ValidationEnabled  bool              // 是否启用校验
    DesensitizeEnabled bool              // 是否启用脱敏
    SafeMode           bool              // 是否启用安全模式
    Concurrency        int               // 并发数（批量转换时使用）
    BatchSize          int               // 批处理大小
    Timeout            time.Duration     // 超时时间
}
```

## 常见组合

### 完整功能

```go
converter := pbmo.NewBidiConverter(PB{}, Model{}).
    WithAutoTimeConversion(true).
    WithValidation(true).
    WithDesensitize(true).
    WithSafeMode(true)
```

### 高性能模式

```go
converter := pbmo.NewBidiConverter(PB{}, Model{}).
    WithAutoTimeConversion(false).
    WithTagMapping(false).
    WithValidation(false)
```

### 自定义 tag

```go
converter := pbmo.NewBidiConverter(PB{}, Model{}).
    WithTagName("protobuf").
    WithTagMapping(true)
```

### 字段映射 + 校验

```go
converter := pbmo.NewBidiConverter(UserPB{}, UserModel{}).
    WithFieldMappings(map[string]string{
        "ID":    "Id",
        "Email": "Mail",
    }).
    WithValidation(true)
```

## 底层 API

### DefaultOptions

```go
options := pbmo.DefaultOptions()
options.ValidationEnabled = true
```

### ApplyOptions

```go
options := pbmo.ApplyOptions(
    pbmo.WithValidation(true),
    pbmo.WithFieldMapping("ID", "Id"),
)
```
