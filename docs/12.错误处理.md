# 12. 错误处理

go-pbmo 定义了类型化错误体系，便于精确识别和处理不同类型的错误

## 错误类型层次

```bash
error
├── ConversionError    // 转换错误
├── NilValueError      // nil 值错误
├── BatchError         // 批量转换错误
├── ValidationError    // 单个校验错误
└── ValidationErrors   // 校验错误集合
    └── []ValidationError
```

## ConversionError

转换过程中发生的错误：

```go
type ConversionError struct {
    Message string
    Cause   error
}
```

```go
err := converter.ConvertPBToModel(pb, &model)
if err != nil {
    if convErr, ok := err.(*pbmo.ConversionError); ok {
        fmt.Printf("转换失败: %s, 原因: %v\n", convErr.Message, convErr.Cause)
    }
}
```

## NilValueError

传入 nil 或无效值时的错误：

```go
type NilValueError struct {
    Message string
}
```

```go
err := converter.SafeConvertPBToModel(nil, &model)
if err != nil {
    if nilErr, ok := err.(*pbmo.NilValueError); ok {
        fmt.Printf("nil 值: %s\n", nilErr.Message)
    }
}
```

## BatchError

批量转换中某个元素失败时的错误：

```go
type BatchError struct {
    Index   int
    Message string
    Cause   error
}
```

```go
err := converter.BatchConvertPBToModel(pbs, &models)
if err != nil {
    if batchErr, ok := err.(*pbmo.BatchError); ok {
        fmt.Printf("元素 %d 失败: %s\n", batchErr.Index, batchErr.Message)
    }
}
```

## ValidationError / ValidationErrors

校验失败时的错误：

```go
type ValidationError struct {
    Field   string
    Message string
}

type ValidationErrors []ValidationError
```

```go
err := validator.Validate(data)
if err != nil {
    if validationErrs, ok := err.(pbmo.ValidationErrors); ok {
        for _, e := range validationErrs {
            fmt.Printf("字段 %s: %s\n", e.Field, e.Message)
        }
    }
}
```

### ValidationErrors 方法

| 方法 | 说明 |
|------|------|
| `Error() string` | 实现 error 接口 |
| `HasErrors() bool` | 是否有错误 |
| `Fields() []string` | 获取所有错误字段名 |

## 错误创建函数

```go
pbmo.NewConversionError(format, args...)  // 创建转换错误
pbmo.NewNilValueError(format, args...)    // 创建 nil 值错误
pbmo.NewBatchError(format, args...)       // 创建批量错误
```

## 最佳实践

### 错误类型判断

```go
err := converter.ConvertPBToModel(pb, &model)
switch e := err.(type) {
case *pbmo.ConversionError:
    log.Printf("转换错误: %v", e)
case *pbmo.NilValueError:
    log.Printf("空值错误: %v", e)
case *pbmo.BatchError:
    log.Printf("批量错误[%d]: %v", e.Index, e)
case pbmo.ValidationErrors:
    for _, ve := range e {
        log.Printf("校验失败 %s: %s", ve.Field, ve.Message)
    }
}
```

### 错误链

`ConversionError` 和 `BatchError` 支持 `Cause` 字段，保留原始错误：

```go
if convErr, ok := err.(*pbmo.ConversionError); ok && convErr.Cause != nil {
    log.Printf("根本原因: %v", convErr.Cause)
}
```
