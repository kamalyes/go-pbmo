# 06. 注册中心

注册中心（Registry）统一管理多个转换器实例，适合多类型转换场景

## 泛型注册（推荐）

泛型 API 利用 `reflect.Type` 做 key + `sync.Map` 缓存，一行注册、按需获取：

### Register — 一行注册

```go
// 自动创建 BidiConverter 并缓存，无需手动 NewBidiConverter
pbmo.Register[UserModel, UserPB]()
pbmo.Register[ProductModel, ProductPB]()
pbmo.Register[OrderModel, OrderPB]()
```

### RegisterWith — 自定义配置注册

```go
pbmo.RegisterWith[UserModel, UserPB](
    pbmo.WithAutoTimeConversion(true),
    pbmo.WithFieldMapping("ID", "Id"),
)
```

### ConverterFor — 获取已注册的转换器

```go
c := pbmo.ConverterFor[UserModel, UserPB]()
c.RegisterTransformer("Name", func(v interface{}) interface{} {
    return strings.ToUpper(v.(string))
})
```

### 泛型转换函数

注册后可直接使用泛型转换函数，无需手动获取转换器：

```go
// 单条转换
pb, _ := pbmo.ToPB[UserModel, UserPB](&UserModel{ID: 1, Name: "张三"})
model, _ := pbmo.FromPB[UserPB, UserModel](&UserPB{Id: 2, Name: "李四"})

// 批量转换
pbs, _ := pbmo.ToPBs[UserModel, UserPB](models)
models, _ := pbmo.FromPBs[UserPB, UserModel](pbs)
```

## 传统注册中心（仍可用）

> ⚠️ 全局便捷函数已标记为 **Deprecated**，建议使用泛型 API

### 创建注册中心

```go
registry := pbmo.NewRegistry()
```

### 注册转换器

```go
// MustRegister — 注册转换器，如果类型已注册则 panic
registry.MustRegister(converter)

// Register — 注册转换器，返回 error
err := registry.Register(converter)
```

### 通过注册中心转换

```go
var model UserModel
err := registry.ConvertPBToModel(pb, &model)

var pb2 UserPB
err = registry.ConvertModelToPB(model, &pb2)
```

### 全局便捷函数（Deprecated）

```go
// Deprecated: 使用 Register[M, P]() 替代
pbmo.MustRegisterConverter(converter)

// Deprecated: 使用 FromPB[P, M](pb) 替代
pbmo.ConvertPBToModel(pb, &model)

// Deprecated: 使用 ToPB[M, P](model) 替代
pbmo.ConvertModelToPB(model, &pb)

// Deprecated: 使用 ConverterFor[M, P]() 替代
converter, err := pbmo.GetConverter(pbType, modelType)
```

## 查询操作

| 方法 | 说明 |
|------|------|
| `Has(pbType, modelType)` | 检查是否已注册 |
| `Lookup(pbType, modelType)` | 查找转换器 |
| `Unregister(pbType, modelType)` | 移除转换器 |
| `Clear()` | 清空所有 |
| `Count()` | 获取已注册数量 |
| `Keys()` | 获取所有 key |

## 多类型管理示例

```go
// 泛型方式（推荐）
pbmo.Register[UserModel, UserPB]()
pbmo.Register[ProductModel, ProductPB]()
pbmo.Register[OrderModel, OrderPB]()

// 直接使用，自动路由到正确的转换器
userPB, _ := pbmo.ToPB[UserModel, UserPB](&userModel)
productPB, _ := pbmo.ToPB[ProductModel, ProductPB](&productModel)
orderPB, _ := pbmo.ToPB[OrderModel, OrderPB](&orderModel)
```

## 迁移对照表

| 旧 API（Deprecated） | 新 API（推荐） |
|----------------------|---------------|
| `NewBidiConverter(PB{}, Model{})` + `MustRegisterConverter(c)` | `Register[M, P]()` |
| `GetConverter(pbType, modelType)` | `ConverterFor[M, P]()` |
| `ConvertPBToModel(pb, &model)` | `FromPB[P, M](pb)` |
| `ConvertModelToPB(model, &pb)` | `ToPB[M, P](model)` |
| `RegisterConverter(c)` | `Register[M, P]()` |

> ⚠️ 全局注册中心适合简单场景，在需要隔离的复杂场景中，建议创建独立的 Registry 实例
