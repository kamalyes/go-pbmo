# 06. 注册中心

注册中心（Registry）统一管理多个转换器实例，适合多类型转换场景

## 创建注册中心

```go
registry := pbmo.NewRegistry()
```

## 注册转换器

### MustRegister

注册转换器，如果类型已注册则 panic：

```go
registry.MustRegister(converter)
```

### Register

注册转换器，返回 error：

```go
err := registry.Register(converter)
if err != nil {
    // 类型已注册
}
```

## 通过注册中心转换

```go
var model UserModel
err := registry.ConvertPBToModel(pb, &model)

var pb2 UserPB
err = registry.ConvertModelToPB(model, &pb2)
```

注册中心根据目标类型自动查找对应的转换器

## 查询操作

| 方法 | 说明 |
|------|------|
| `GetConverter(typeName)` | 获取转换器 |
| `HasConverter(typeName)` | 检查是否已注册 |
| `ListConverters()` | 列出所有已注册的转换器 |
| `RemoveConverter(typeName)` | 移除转换器 |
| `Clear()` | 清空所有 |

## 全局便捷函数

go-pbmo 提供包级别的全局注册中心，无需手动创建 Registry：

```go
// 注册到全局注册中心
pbmo.MustRegisterConverter(converter)

// 全局转换
pbmo.ConvertPBToModel(pb, &model)
pbmo.ConvertModelToPB(model, &pb)

// 查询
pbmo.GetGlobalConverter("UserModel")
pbmo.HasGlobalConverter("UserModel")
```

> ⚠️ 全局注册中心适合简单场景在需要隔离的复杂场景中，建议创建独立的 Registry 实例

## 多类型管理示例

```go
userConverter := pbmo.NewBidiConverter(UserPB{}, UserModel{})
productConverter := pbmo.NewBidiConverter(ProductPB{}, ProductModel{})
orderConverter := pbmo.NewBidiConverter(OrderPB{}, OrderModel{})

registry := pbmo.NewRegistry()
registry.MustRegister(userConverter)
registry.MustRegister(productConverter)
registry.MustRegister(orderConverter)

// 自动路由到正确的转换器
var user UserModel
registry.ConvertPBToModel(userPB, &user)

var product ProductModel
registry.ConvertPBToModel(productPB, &product)
```
