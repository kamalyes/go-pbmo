# 09. 安全转换

go-pbmo 集成 `go-toolbox/safe`，提供 nil 安全的转换和字段访问能力，避免运行时 panic

## SafeConverter

```go
converter := pbmo.NewSafeConverter(UserPB{}, UserModel{})
```

### 安全 PB → Model

```go
var model UserModel
err := converter.SafeConvertPBToModel(pb, &model)
// 自动处理 nil 值，不会 panic
```

如果 `pb` 为 nil 或无效，返回 `NilValueError` 而非 panic

### 安全 Model → PB

```go
var pb UserPB
err := converter.SafeConvertModelToPB(model, &pb)
```

## 安全字段访问

### SafeFieldAccess

链式安全访问嵌套字段，避免中间层 nil 导致 panic：

```go
converter := pbmo.NewSafeConverter(UserPB{}, UserModel{})

sa := converter.SafeFieldAccess(user, "Profile", "Avatar")
if sa.IsValid() {
    avatar := sa.String()
}
```

### SafeGetField

安全获取单个字段值：

```go
sa := pbmo.SafeGetField(user, "Name")
if sa.IsValid() {
    name := sa.String()
}
```

### SafeGetNestedField

通过路径访问嵌套字段：

```go
sa := pbmo.SafeGetNestedField(user, "Profile.Avatar.URL")
if sa.IsValid() {
    url := sa.String()
}
```

## 安全批量转换

```go
result := converter.SafeBatchConvertPBToModel(pbs, &models)

// 自动跳过 nil/零值元素
for _, item := range result.Results {
    if !item.Success {
        log.Printf("元素 %d 转换失败: %v", item.Index, item.Error)
    }
}
```

与 `BidiConverter.SafeBatchConvertPBToModel` 不同，`SafeConverter` 的安全批量转换额外使用 `go-toolbox/safe` 检查每个元素是否为 nil/零值，在转换前就跳过无效元素

## SafeAccess 链式调用

`go-toolbox/safe` 的 `SafeAccess` 支持链式调用：

```go
sa := safe.Safe(user).
    Field("Profile").
    Field("Settings").
    Field("Theme")

if sa.IsValid() {
    theme := sa.String()
} else {
    theme = "default"
}
```

## 错误类型

安全转换使用 `NilValueError` 表示 nil 值错误：

```go
err := converter.SafeConvertPBToModel(nil, &model)
if err != nil {
    if nilErr, ok := err.(*pbmo.NilValueError); ok {
        fmt.Println("nil 值错误:", nilErr)
    }
}
```
